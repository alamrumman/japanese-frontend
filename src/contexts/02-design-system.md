# 02 — Design System

> Version 2.0

---

# Purpose

The Design System defines the visual language of the website.

It exists to ensure every page feels like it belongs to the same experience.

Users should never feel they have navigated to another website.

The system prioritizes consistency over novelty.

Every future component should inherit these rules.
