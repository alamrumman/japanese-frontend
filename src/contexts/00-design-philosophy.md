# Purpose

This document establishes the visual philosophy, emotional direction and creative principles that define the project.

It exists to ensure every future design decision feels consistent.

The purpose is not to build an attractive website.

The purpose is to create an experience that communicates craftsmanship, elegance and authenticity.

Everything that follows in this documentation inherits the rules defined here.

---

# The Core Idea

This is **not** a restaurant website.

It is a digital interpretation of walking into an intimate omakase restaurant in Tokyo.

The website should not feel commercial.

It should feel personal.

Quiet.

Slow.

Confident.

Every screen should communicate that the restaurant cares deeply about every detail.

That same care should exist inside the interface.

---

# Design Statement

Design should disappear.

Users should notice the experience before they notice the interface.

Nothing should exist because it looks impressive.

Everything should exist because it improves storytelling.

If an element can be removed without reducing the experience—

remove it.

## Editorial Composition

The layout resembles a magazine rather than a landing page.

There are no obvious sections.

Instead there are visual chapters.

Photography flows into typography.

Whitespace separates content naturally.

Users discover content instead of scanning boxes.

---

## Quiet Luxury

The interface never attempts to prove it is expensive.

It simply feels expensive.

This comes from restraint.

Not decoration.

Characteristics include

• restrained color palette

• minimal borders

• subtle typography

• high quality imagery

• generous spacing

• almost invisible animations

• balanced composition

---

## Japanese Minimalism

Japanese design values subtraction.

Instead of asking

"What else can be added?"

Ask

"What can still be removed?"

Every page should contain breathing room.

Large empty areas are intentional.

Negative space becomes part of the composition.

---

# Editorial Design Language

Imagine every page as a magazine spread.

Typography.

Photography.

Whitespace.

Composition.

These four elements should build every page.

Never rely on decorative graphics.

Never rely on colorful UI.

The experience should feel printed before it feels digital.

---

# Craftsmanship over Marketing

Traditional restaurant websites immediately attempt to sell food.

This website should first communicate craftsmanship.

Users should understand

who made the food

before

what food is served.

Trust precedes appetite.

---

# Photography Philosophy

Photography is not decoration.

Photography tells the story.

Images should appear professionally captured.

Natural lighting.

Soft shadows.

Warm highlights.

Real textures.

Never use obviously staged stock photography.

Avoid overly saturated colors.

Every photograph should feel timeless.

---

# Typography Philosophy

Typography should communicate hierarchy before color.

Headings should command attention.

Body copy should disappear.

Japanese serif typography should carry emotion.

Sans-serif typography should carry information.

Large typography should create atmosphere.

Small typography should create clarity.

---

# Layout Philosophy

The layout should never appear generated.

Avoid repetitive grids.

Avoid obvious templates.

Avoid identical cards.

Instead create rhythm.

# Visual Rhythm

Every page should alternate between

Heavy

↓

Light

↓

Heavy

↓

Light

↓

Heavy

This prevents visual fatigue.

Do not create consecutive image-heavy sections.

Likewise

Do not create consecutive text-heavy sections.

Every section should balance the previous one.

# Visual Hierarchy

Every screen should answer these questions within three seconds.

Where am I?

What is important?

Where should I look next?

Hierarchy should come from

Typography

↓

Spacing

↓

Photography

↓

Motion

Never from color.

---

# Color Philosophy

Black is not simply a background.

Black represents silence.

Gold represents craftsmanship.

White represents honesty.

Red represents restraint.

Use red sparingly.

When users notice red

it should immediately feel intentional.

---

# Motion Philosophy

Motion exists to guide attention.

Not entertain.

Users should rarely notice animations.

Instead

they should notice clarity.

The best animation is the one users never consciously remember.

---

# Component Philosophy

Every component should feel individually designed.

Buttons.

Cards.

Forms.

Navigation.

Footer.

Nothing should feel copied.

Nothing should feel generated.

If multiple components share the exact same proportions—

consider redesigning one.

---

# Interaction Philosophy

Interactions should feel physical.

Buttons should have weight.

Navigation should feel tactile.

Hover states should communicate intention.

Scrolling should feel smooth but restrained.

The interface should never appear playful.

It should appear confident.

---

# Mobile Philosophy

Mobile is not a compressed desktop.

Mobile is its own experience.

Layouts should be redesigned.

Typography should increase.

Spacing should increase.

Images should become immersive.

Touch interactions replace hover interactions.

Users should feel the website was originally designed for mobile.

---

# Accessibility Philosophy

Accessibility should never reduce beauty.

Accessibility should improve clarity.

High contrast.

Visible focus states.

Keyboard navigation.

Reduced motion support.

Readable typography.

These should be integrated naturally.

Never treated as afterthoughts.

---

# Performance Philosophy

Luxury websites should feel effortless.

Performance contributes directly to perceived quality.

Animations should never block rendering.

Images should never delay interaction.

Pages should feel immediately responsive.

The website should appear calm because it is technically efficient.

---

# AI Detection Checklist

Before approving any design ask

Does this resemble a template?

Does this feel symmetrical?

Is every section using identical spacing?

Are cards repeated?

Are shadows excessive?

Is animation trying too hard?

Would this still look premium without motion?

Would a designer believe another designer created this?

If any answer is yes

redesign.

---

# Final Design Principle

Do not design for attention.

Design for memory.

Visitors should not remember the interface.

They should remember how the website made them feel.

That feeling should be

Quiet.

Premium.

Authentic.

Timeless.

Japanese.
